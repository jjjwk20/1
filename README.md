# 信号处理实验 - 卷积计算

## 项目描述
本项目演示了如何在 MATLAB 中实现信号的**卷积运算**，使用离散近似的方法计算连续信号的卷积结果。

## 实验内容

### 目标
计算两个信号的卷积：$f(t) * h(t)$
- **输入信号**：$f(t) = 2[\varepsilon(t) - \varepsilon(t-2)]$（矩形脉冲）
- **冲激响应**：$h(t) = e^{-t}\varepsilon(t)$（指数衰减）
- **输出结果**：卷积结果 $y(t) = f(t) * h(t)$

### 核心思想
- 使用离散时间近似连续信号
- 利用 MATLAB 的 `conv()` 函数计算离散卷积
- 乘以时间步长 `dt` 将离散结果转换为连续卷积的近似值

## 文件说明

## 使用方法

1. **打开 MATLAB**
2. **运行脚本**
   ```matlab
   experient01.m
   ```
3. **查看结果**：程序将生成一个包含 3 个子图的图形窗口：
   - 第1幅：输入信号 $f(t)$
   - 第2幅：冲激响应 $h(t)$
   - 第3幅：卷积结果 $y(t)$

## 参数设置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `dt` | 0.01 | 时间步长（越小结果越精确） |
| `t` | -1:dt:5 | 时间轴范围 |

## 关键 MATLAB 函数

- `conv(f, h)`：计算两个向量的卷积
- `exp(-t)`：计算指数衰减
- `plot()`：绘制图形
- `subplot()`：创建子图
